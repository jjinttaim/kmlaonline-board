# KMLA Online

KMLA Online (aka hotchoco) 은 민족사관고등학교 과학기술부가 관리하는 민사고의 학생 커뮤니티입니다. 공지사항, 게시판, 채팅 및 학습 자료 공유 등의 기능을 가집니다.

KMLA Online (aka hotchoco) is a school community website of KMLA which serves Announcements, Boards, Chats and Study Materials.

## Getting Started

1. repo 복제

```bash
git clone https://github.com/KMLA-Department-of-SW-and-Tech/hotchoco.git
```

2. 의존성 설치

```bash
cd hotchoco && npm i
```

3. 개발 서버 실행

```bash
npm run dev
```
